It's very easy to make some words **bold** and other words *italic* with Markdown. You can even [link to Google!](http://google.com)
# this is Header1
## this is Header2
### this is Header3
#### this is Header4
##### this is Header5
###### this is Header6
  *this text will be italic*  
  **this test will be bold**
